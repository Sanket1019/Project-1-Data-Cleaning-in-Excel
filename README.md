# 📊 E-commerce Sales Data Cleaning using Microsoft Excel

This project was completed as part of my Data Analytics learning journey to practice real-world data cleaning techniques using Microsoft Excel.

---

## 📌 Project Overview

The objective of this project was to inspect an E-commerce Sales Dataset, identify potential data quality issues, validate the data, and prepare it for Exploratory Data Analysis (EDA).

---

## 🎯 Objectives

- Check for missing values
- Identify duplicate records
- Validate data types
- Verify calculated fields
- Improve dataset readability
- Prepare the dataset for analysis

## 🧹 Data Cleaning Process

### ✅ Missing Value Analysis
- Checked all columns for missing values.
- Blank values in the **Coupon Code** column were retained because they represent orders where no coupon was applied.

### ✅ Duplicate Check
- Checked the complete dataset for duplicate records.
- No duplicate records were found.

### ✅ Data Type Validation
Verified that:
- Date follows the **YYYY-MM-DD** format.
- Quantity contains whole numbers.
- Unit Price contains decimal values.
- Total Price contains decimal values.

### ✅ Total Price Verification
Random records were verified by calculating:

**Quantity × Unit Price = Total Price**

The calculated values matched the dataset.

### ✅ Dataset Formatting
- Converted the dataset into an Excel Table.
- Applied filters to all columns.
- Improved header formatting.
- Added a **Data Cleaning Summary** worksheet for documentation.
